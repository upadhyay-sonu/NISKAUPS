API KEYS CONFIGURATION
================================================================================

If you want to use LIVE APIs instead of fallback data:

In .env file, add:
NEWSAPI_KEY=your_key_from_newsapi.org
GNEWS_KEY=your_key_from_gnews.io
GUARDIAN_KEY=your_key_from_theguardian.com
NYTIMES_KEY=your_key_from_nytimes.com

If API keys are missing:
✅ System uses cached data (from last 10 min)
✅ If no cache, system uses fallback JSON data
✅ News still displays (never empty)
